Big Models made possible

Fast Storage, On-demand Loading and Unloading of your large scale EMF Models

NeoEMF is an AtlanMod research team project that aims at handling large EMF models in an efficient and scalable way. It is based on our work on Neo4EMF, a scalable and graph based backend for persisting EMF models. NeoEMF is a multi-backend framework developped to allow an easy integration of custom backends depending on user needs.

Github Download zip Update site

Java code generation

Given any ecore model, NeoEMF provides you with an adapted generic java code generation usable with several backends

On-demand loading

Light-weight loading of massive queries by only instantiating accessed EObjects

Multiple Backends

NeoEMF provides two backend implementations to fit your needs: a Graph backend (Blueprints/Neo4j) and a Map (MapDB) one.
